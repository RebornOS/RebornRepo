aur

AUR Repository featuring:

  • Automated builds and updates using gitlab.com
  • Daily builds of git packages
  • Repository updates via rss (atom)
  • Weekly repository checks to detect packages requiring a rebuild
  • Packages for any, arm, armv6h, armv7h, aarch64 and x86_64 architecture
# Adding key using keyring 
wget https://repo.itmettke.de/aur/aur-archlinux/aurarchlinux-keyring.pkg.tar.xz 
pacman -U aurarchlinux-keyring.pkg.tar.xz 

# Adding key using recv-key 
pacman-key --recv 7448C890582975CD 
pacman-key --lsign 7448C890582975CD 

# Adding Repository 
nano /etc/pacman.conf 
[aur-archlinux]

aur-archive

Archive containing older package versions from the aur repository

  • Obsolete packages are kept up to 2 month
  • .latest. files always point to the latest version
# Using downgrade 
pacman -S downgrade 
export DOWNGRADE_ALA_URL="https://repo.itmettke.de/aur-archive" 
downgrade downgrade 

# Using agetpkg-git 
pacman -S agetpkg 
export ARCHIVE_URL=https://repo.itmettke.de/aur-archive/packages/.all/ 
agetpkg -l webkitgtk
